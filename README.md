# Portfolio Website

A personal portfolio website built to showcase my projects and skills.

## Built With

- React + TypeScript
- Vite
- Tailwind CSS
- CSS (Glassmorphism styling)

## What I Learned

- Setting up a React + TypeScript project with Vite
- Component-based architecture — breaking UI into reusable pieces
- Using CSS variables for consistent theming across components
- Glassmorphism design — backdrop-filter, rgba backgrounds, border opacity
- Building a horizontal card slider with pixel-accurate translateX offsets
- Measuring DOM elements with useRef to calculate dynamic card widths
- Keyboard event listeners in React (arrow key navigation)
- IntersectionObserver API for scroll-reveal animations
- Typewriter effect using useEffect and recursive state updates
- Responsive design — different layouts for mobile and desktop
- Animated blobs using CSS keyframes and custom properties
- Deploying a Vite app to Vercel
